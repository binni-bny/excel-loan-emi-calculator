# Loan & EMI Financial Planning Calculator

Interactive Excel loan/EMI calculator with amortization schedule, loan comparison, and dashboard — powered by financial functions and real-world data.

## Overview
An Excel-based loan calculator that computes EMI, generates a full 
60-month amortization schedule, and compares multiple loan offers 
side by side.

## Features
- EMI calculation using PMT function
- Month-by-month amortization schedule (principal vs interest breakdown)
- Loan comparison tool for evaluating different rate/tenure options
- Interactive dashboard with charts

## Skills Demonstrated
- Financial functions (PMT)
- Absolute cell referencing
- Cross-sheet formulas
- Data visualization (pie chart, line chart)
- Real-world dataset integration (Kaggle personal finance data)

## Dataset
Synthetic Personal Finance dataset (Kaggle) used to validate 
formulas against real EMI values.
